# 市场分析技能

## 描述
该技能用于分析股票市场数据，包括价格趋势、交易量、技术指标等。

## 输入参数
- symbol: 股票代码
- period: 时间周期（如 1d, 1w, 1m）
- indicators: 技术指标列表（如 RSI, MACD, SMA）

## 执行步骤
1. 获取股票的历史价格数据
2. 计算指定的技术指标
3. 分析价格趋势和指标信号
4. 生成分析报告

## 输出格式
```json
{
  "symbol": "股票代码",
  "period": "时间周期",
  "indicators": {
    "RSI": 数值,
    "MACD": 数值,
    "SMA": 数值
  },
  "trend": "趋势分析",
  "signal": "买入/卖出/持有信号",
  "analysis": "详细分析报告"
}
```
